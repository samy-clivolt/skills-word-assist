# Analyse Depenses

Ce skill aide a analyser un portefeuille d'achats et proposer des gains rapides.

## Quand l'utiliser

- Tu veux identifier des categories prioritaires
- Tu prepares un plan d'economies
- Tu dois segmenter les fournisseurs

## Etapes

1. Rassembler les donnees (depenses, fournisseurs, contrats)
2. Segmenter par categorie et volume
3. Identifier les axes d'optimisation
4. Proposer un plan d'action avec estimations
